# Customer Churn Prediction System

##  Project Overview
The Customer Churn Prediction System is a machine learning project that predicts whether a customer is likely to leave a telecom service. The goal of this project is to help businesses identify at-risk customers and take proactive measures to improve retention.

##  Features
- Predicts customer churn using machine learning
- Uses Random Forest Classifier for prediction
- Interactive web application built with Streamlit
- Real-time prediction based on user input

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Git & GitHub

##  Dataset
- Telco Customer Churn Dataset
- Dataset contains customer demographic, service usage, and billing information

## ⚙️ Project Workflow
1. Data preprocessing (handling missing values, encoding, scaling)
2. Model training using Random Forest algorithm
3. Model evaluation
4. Deployment using Streamlit web application

## ▶️ How to Run the Project

Install the required libraries, train the machine learning model, and run the Streamlit application using the following commands:

```bash
pip install pandas scikit-learn streamlit joblib
python train_model.py
streamlit run app.py
